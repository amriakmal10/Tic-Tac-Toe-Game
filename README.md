# Tic-Tac-Toe

A classic Tic-Tac-Toe game implemented with **React**. This project provides a simple, interactive user interface to play the game in a browser.

## Features
* **Interactive Gameplay**: Players alternate turns between 'X' and 'O'.
* **Win/Draw Detection**: Automatically announces the winner or declares a draw when the grid is full.
* **Game Reset**: A dedicated reset button to start a new game at any time.
* **State Management**: Built with React's `useState` to track board updates and game status.

## Tech Stack
* **React 18**
* **CSS3** (Grid/Flexbox for layout)
* **Babel** (In-browser JSX compilation)

## Getting Started
1. Clone this repository to your local machine.
2. Open `index.html` using a local web server (The **Live Server** extension in VS Code) to avoid browser CORS restrictions.
3. Play the game directly in your browser.
